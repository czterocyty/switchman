A/B Test Microservice

Configuration and REST API for (cross-module) A/B tests.

The test configurations can be modified via the web user interface and the REST-API. When using the API test descisions can be stored and retrieved.

When you like to call the REST Interface you will need credentials that must be passes via HTTP Basic Authentication!
